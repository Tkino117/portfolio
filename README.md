# Portfolio

Personal portfolio site built with Next.js and deployed on Cloudflare.

## Tech Stack

- [Next.js](https://nextjs.org) (App Router)
- [Tailwind CSS](https://tailwindcss.com)
- [shadcn/ui](https://ui.shadcn.com)
- [Cloudflare Pages](https://pages.cloudflare.com) via [OpenNext](https://opennext.js.org/cloudflare)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it locally.

## Deploy

```bash
npm run deploy
```
